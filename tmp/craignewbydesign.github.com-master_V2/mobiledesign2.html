Purpose

The purpose of this app is simply to list recipes, ingredients, and steps. It also uses the Google Maps API to locate ingredients that you may not have at your local grocery store.This app was created to show the UI principles at a high level block mockup. The goal was to create an app that flows following purely the principles of user interface design. The app flows from page to page using only layout, color, and transitions.

Technologies Used

HTML 5, CSS 3, JavaScript
